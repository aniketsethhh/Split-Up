# Split-Up 💸

The smartest way to split expenses with friends.

## 🌐 Live Demo
[https://split-up-sage.vercel.app](https://split-up-sage.vercel.app)

## ✨ Features
- Track shared expenses with friends and groups
- Split bills equally, by percentage, or exact amounts
- Settle up and track payments
- Dark/Light mode
- Monthly spending insights

## 🛠️ Tech Stack
- **Frontend**: Next.js 16, Tailwind CSS, shadcn/ui
- **Backend**: Convex
- **Auth**: Clerk
- **Email**: Resend + Inngest
- **AI Insights**: Google Gemini

## 🚀 Getting Started

1. Clone the repo
2. Install dependencies:
```bash
   cd my-app
   npm install
```
3. Set up environment variables in `.env.local`
4. Run the development server:
```bash
   npx convex dev
   npm run dev
```
